# Changelog — JVO Agent Manager Prototype

Список всех изменений, внесённых в прототип после базового коммита (`091d104`).

---

## Исправления и доработки существующего флоу

- **Подтверждение корректировок** — кнопка «Подтвердить» в сценариях теперь убирает корректировку и показывает тост «Корректировка подтверждена»
- **Тоггл «Публиковать без подтверждения»** — при выключении открывает модальное окно с предупреждением
- **Info-попап** — кнопка `ⓘ` рядом с тогглом показывает всплывающее объяснение
- **Inline-редактирование названия плана** — карандаш (Edit3) открывает поле ввода прямо в заголовке
- **«Опубликовать все» / «Пропустить все»** — модальное подтверждение в табах «Сценарии» и «События»
- **Фильтр событий** — шестерёнка в таблице событий открывает фильтр по статусу
- **Исправлено название сервиса** — «Джвио» → «Дживио» везде в тексте

---

## Сайдбар

- **Новый дизайн по Figma** — выезжает при наведении, фиксированная ширина-заглушка 56px в лейауте
- **Полная структура навигации** — логотип, топ-навигация (5 пунктов), аккордеон «Автоматизация» с подпунктами, PRO-бейджи на SEO и Логистике, аватар внизу
- **«Планы продаж» кликабелен** — навигация работает с любого экрана через `onNavigate` prop

## Экран чата

- **Убрана кнопка «Давай шурши»** — шторка анализа появляется автоматически через 700 мс после завершения диалога

## Шторка анализа (AnalysisPanel)

- **Новый дизайн по Figma** — вкладки «Анализ | Стратегия | Цели | Сценарии»
- **Вкладка «Анализ»** — строки рассуждений агента + инсайт-карточки + кандидаты сценариев с анимацией схлопывания отклонённых
- **Вкладка «Стратегия»** — описание стратегии восстановления маржи на основе реальных данных
- **Вкладка «Цели»** — 4 KPI-карточки (цель маржи, целевая маржинальность, стартовая точка, прирост) + компактный график роста маржи по неделям + таблица недельного плана
- **Вкладка «Сценарии»** — список 5 сценариев в стиле макета

## Дашборд — вкладка «План»

- **Название плана** — «Вся матрица Wildberries — 60 дней (06.05–04.07)»
- **Описание** — стратегия восстановления маржи (цена → акции → реклама)
- **Тег статуса** — «Планом управляет Агент» (фиолетовый) вместо «Активен» (зелёный)
- **Метрики** — переведены с выручки на маржу: цель 42 330 ₽, текущая 14 100 ₽, выполнение 33%
- **Progress bar** — показывает накопленную маржу к цели
- **Правый блок метрик** — маржинальность сейчас 13,1% с прогрессом к цели 18,5%
- **Подпись** — «Обновлён 25.05 · Следующее обновление через неделю»
- **График** — маржинальность % по неделям, план (пунктир) vs факт (сплошная), нед. 1–4 с фактом
- **Плашка действий агента** — под графиком: итого 44 действия / Отзывы 22 / Кластеры 8 / Изм. цены 14
- **Таблица план/факт** — 9 недель, колонки: маржа план, маржа факт, % план, % факт, отклонение; текущая неделя (нед. 4) подсвечена

## Экран «Планы продаж»

- **Empty state** — иконка, текст «Нет ни одного плана», большая кнопка «Создать план продаж»
- **Master-detail лейаут** — список слева (384px) + превью справа
- **Строка в списке** — зелёная пульсирующая точка статуса, название (без обрезания), 4 метрики: выполнение / отклонение / план ₽ / факт ₽
- **Сортировка** — дропдаун: по умолчанию / по названию / по отклонению / по выполнению
- **Стрелка по ховеру** — `→` появляется рядом с названием при наведении, открывает план напрямую
- **Превью плана** — тулбар (название + кнопка «Открыть план»), статус + дата обновления, описание, метрики, прогресс-бар, блок маржинальности, компактный график
- **«Открыть план»** — кнопка в тулбаре превью, всегда видна без скролла
- **DemoPanel** — добавлены кнопки «Планы: пусто» и «Планы: список»
